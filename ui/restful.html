Mini-Tutorial

Searching for entries with a particular tag is easy as ABC. Just set the URL as http://localhost:5000/q and append it with only one GET parameter tag. This should have the value of the tag you are searching for.

Inserting a new entry is simple as well. Append to the URL with knowledge and enter into the input area a JSON object with two key value pairs: namely "content" with the corresponding content of the entry, and the "tags" with the tags that it should be associated with as a CSV string. For example: { "content": "Classified content", "tags": "gov,nsa" }

Deleting an entry ain't no rocket surgery as well. Merely append to the URL /knowledge/id , where id is the id of the entry to be deleted (can be found out by using a search query).

Finally, updating can be performed by following the same instructions as inserting but using /knowledge/id as the URI instead, where id is the id of the entry to be updated.
